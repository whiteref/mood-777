# Design System: Mood-777 (Minimal Luxury)

## Design Identity
- **Pattern:** Golden Grid (우아한 대칭과 황금비율 기반)
- **Style:** Neo-Classical / Minimal Luxury
- **Concept:** "The Secret Ritual of Light and Shadow"

## Color Palette (Original Maintained)
- **Primary Pink (#FF7EAD):** Main Theme Color, Highlights
- **Soft Background (#FDFCFB):** Main Background, Canvas
- **Warm Pink (#FF9A9E):** Gradient Start, Primary Buttons
- **Lavender Pink (#FECFEF):** Gradient End, Highlights
- **Slate-700 (#334155):** High Contrast Text (For readability)
- **Stone-400 (#A8A29E):** Muted Text, Metadata

## Typography
- **Headings:** [Cormorant Garamond](https://fonts.google.com/specimen/Cormorant+Garamond) (Serif)
  - Weight: 600 (Semi-bold) for main titles
- **Body:** [Montserrat](https://fonts.google.com/specimen/Montserrat) (Sans-serif)
  - Weight: 400 (Regular), 500 (Medium) for UI labels

## Visual Effects
- **Backdrop Blur:** `backdrop-blur-xl bg-white/70`
- **Transitions:** `transition-all duration-500 ease-in-out`
- **Shadows:** Soft, deep shadows for depth (`shadow-2xl shadow-stone-200`)

## Icons
- **Library:** Lucide React
- **Stroke Width:** 1.5px (Thin, Elegant)

## Animation Guidelines
- **Entrance:** Fluid slide-up with opacity (0.8s)
- **Interaction:** Subtle scaling (1.02) and color shifting.
- **Loading:** Minimal geometric spinners or skeleton UI.

## Anti-Patterns
- ❌ No vibrant or neon colors (Keep it muted and premium)
- ❌ No thick borders (Use shadow and contrast instead)
- ❌ No emojis as UI icons (Use SVG/Lucide)
